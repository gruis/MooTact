How to use
----------
 Instantiate a MooTact object and call show();
#JS

	new MooTact( { title : 'MooTact Contact Form', 
			               onSuccess: function(){  $("notify").set('html', 'E-mail sent ... Thank you!' ); } 
							}).show();
